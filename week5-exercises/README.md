# CIS18B Week 5 – Collections, Iterators, and Data Structures

## How to Run

1. Navigate to a mini-exercise folder:
   cd week5-exercises/mini01-collection-selection-clinic

2. Compile:
   javac src/main/java/edu/norcocollege/cis18b/week5/mini01/*.java

3. Run:
   java -cp src/main/java edu.norcocollege.cis18b.week5.mini01.CollectionSelectionClinic

Repeat for each mini-exercise by changing the folder and class name.

## Summary

This assignment demonstrates:
- List, Set, Map, Queue, and Deque usage
- Iterators and ListIterator
- Custom Iterable and Iterator patterns
- Collection behavior and tradeoffs
